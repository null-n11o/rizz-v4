## 概要

ストリートナンパの実績を記録・分析するためのモバイルアプリケーション開発プロジェクト。

声かけ数、連絡先獲得数、即日デート数、即数などの基本指標を記録し、日次/週次/月次/年次での分析機能を提供する。

## 対象者

- ストリートナンパを行う個人
- ナンパスクールの運営者

## 現在の課題

現在は手書きのメモやスプレッドシートでの管理が主流で、体系的なデータ管理・分析ツールは存在していない。

## 解決の方向性

- スマートフォンアプリで簡単に記録できるUI/UX設計
- 自動集計による統計分析機能の実装
- グラフやチャートによる視覚的なデータ表示
- 目標設定と達成度管理機能の提供

## MVP機能一覧と実装状況

### ✅ 実装済み

1. **基本カウンター機能**
   - ワンタップでの声かけカウント記録
   - ワンタップでの連絡先ゲットカウント記録
   - ワンタップでの即日デート記録
   - ワンタップでの即(sex)記録
   - カウントのデクリメント（取り消し）機能

2. **目標設定機能**
   - 期間別目標設定（日・週・月・年）
   - ホーム画面でのプログレスバー表示

3. **認証機能**
   - メールアドレス & パスワードでサインアップ
   - メールアドレス & パスワードでサインイン
   - ログアウト

4. **設定機能**
   - ユーザーネーム設定・変更
   - X(旧Twitter)アカウントURL 記入・変更
   - パスワード変更
   - テーマ切替（ライト/ダーク）
   - 言語切替（日本語/英語）

5. **オフライン対応**
   - AsyncStorageによるローカルキャッシュ
   - オフライン時の変更キュー
   - オンライン復帰時の自動同期

### ❌ 未実装（MVP残タスク）

1. **統計画面グラフ**（画面は存在するがプレースホルダー）
   - 日次/週次/月次/年次のグラフ表示
   - 成功率（連絡先獲得率、即率）の計算・表示

2. **詳細情報登録**
   - 場所 (game_area)・時間 (game_time) の入力UI

3. **プロフィール画像**
   - Supabase Storageへのアップロード・表示機能

### 🔮 プレミアム機能（将来計画）

1. **高度な分析**
   - 時間帯別・場所別の成功率分析

2. **コミュニティ機能**
   - 全体統計情報の表示
   - ランキング機能

3. **データ管理**
   - CSVエクスポート

## ブランドカラー

Rizzアプリケーションのブランドカラーは、高級感とカリスマ性を表現するために厳選されたカラーパレットを使用します。ダークをベースに、ゴールドとシルバーのアクセントを効果的に配置することで、洗練された男性的な魅力を演出します。

### カラーパレット

| 役割 | 名称 | カラーコード |
|------|------|------------|
| 主要背景 | リッチブラック | `#0A0F23` |
| ロゴ・見出しハイライト | アンティークゴールド | `#C09E5C` |
| ヘッダー/フッター背景 | ダークヘッダー | `#080C1A` |
| カード・ボタン背景 | チャコールグレー | `#36454F` |
| 重要アクション | バーガンディレッド | `#800020` |
| 枠線・装飾 | プラチナシルバー | `#E5E4E2` |
| グラフ・プログレス終端 | メタリックゴールド | `#D4AF37` |

- **基本テキスト**: 白 `#FFFFFF`
- **プログレスバーグラデーション**: `#0A0F23` → `#1A2342` → `#D4AF37`
